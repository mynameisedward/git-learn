Git — система контроля версий



Возможности:
- откат до предыдущей версии проекта
- просмотр истории изменений
- совместная работа



Git интегрирован во все редакторы кода



Индекс — это такая область куда попадают файлы, которые подготавливаются к коммиту

Добавить файлы в индекс можно командой git add имя_файла/либо_. (точка добавляет все файлы в индекс кроме исключений в .gitignore)

Зачем добавлять файлы в индекс? Разве нельзя просто сделать коммит и добавить все файлы в репозиторий?
Это нужно для того чтобы, например мы изменили один файл и хотим добавить именно его, а не весь проект 



Когда мы делаем коммит, в коммит попадают именно те файлы которые были в индексе

Размер коммита бывает разным:
- маленьким, если мы исправляем только несколько опечаток
- большим, если внедряем новую функциональность

Главное в коммите — атомарность, то есть он должен выполнять только одну задачу
